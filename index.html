Graduate School Projects

Monte Carlo RL Racetrack Simulation

Developed a Monte Carlo Agent to optimize performance on a simulated racetrack with slippage, using reinforcement learning. The project includes custom track building and performance visualizations.

AI Genetic Algorithm

Implemented a genetic algorithm to solve the backpacking problem, optimizing item selection to maximize total value without exceeding a weight limit of 250 units. Each item is characterized by its weight and importance value.

Sudoku-Solver-CSP

The solver implements constraint satisfaction problem (CSP) principles, ensuring that the puzzle adheres to the fundamental rules of Sudoku. The solver uses recursive backtracking, which tries numbers 1-9 in empty cells and backtracks when it encounters a conflict.

Pre-Graduate School Projects

FM Radio Landscape

FCC data on ownership and geolocation of FM stations, geospatial data for each congressional district, and US Census data combine to create an ‘FM Landscape.’ Useful for targeted messaging strategies by voting district or population centers. Ongoing project in development.

Stellar Object Classification

Developed a random forest classifier using data on light wavelengths and redshift from over 2 million labeled stellar objects, queried via SQL. Achieved 99% accuracy on unseen data.
