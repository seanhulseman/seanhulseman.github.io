Some of my recent data science projects

Stellar Object Classification

Created a random forest classifier with data on certain wavelengths of light from 500,000 labeled stellar objects obtained a via SQL query of the Sloan Digital Sky Survey. On unseen data, the model's predictions were 99.4% accurate.

Ames Housing Project

Built an compared various regression models' performance on predicting housing prices from about 80 variables describing the house and its condition/quality. I built methods to standardize data preperation to quickly generate predictions on unseen data.

FM Radio Landscape

The radio landscape is an idea I came up with on a road trip in August 2022. I wanted to understand, as I drove away from familiar cities, how the radio options were so vastly different over a few dozen miles of highway. The FCC FM Radio Query was used to collect the data and label commercial vs educational stations. There are many directions one can take a project with this data. I encourage anyone to make their own projects using the data collected jan/26/2024 'FM_stations_labeled.csv'
