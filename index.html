Stellar Object Classification

Created a random forest classifier with data on certain wavelengths of light from 500,000 labeled stellar objects obtained a via SQL query of the Sloan Digital Sky Survey. On unseen data, the model's predictions were 99.4% accurate.

Subreddit Classifier

Scraped 1000 posts each from two subreddits using the PRAW Reddit API. I was able to accurately classify the posts as coming from Chicago or Boston 82% of the time. Skills used: CountVectorization, TD-IDF, GridSearch, Pipelines, SVCs, Logistic Regression

Ames Housing Project

Built a model to predict sale prices of homes using nearly 80 features. I built methods to standardize my EDA, cleaning, dealing with missing valeus, one hot encoding categorical variables, etc. to quickly generate predictions on messy unseen data.

FM Radio Landscape

Created a geospatial dataframes from census and FCC data that maps all FM radio transmitters and all cities and towns over 10,000 people. I ended up creating a geodataframe that can be used to find what stations any population in the US can listen to. This was my first 'data science' project and I am meaning to revisit it since I have gotten better at this over the past year!
