FCC Radio Landscape Project

This project builds an end-to-end pipeline that turns FCC quarterly “Issues & Programs” PDFs into structured data, then connects station coverage, district/precinct geography, and NLP-derived signals to study what is discussed, where, and when.

Figure 1 — Coverage Universe

Towers and modeled broadcast ranges for stations with available FCC filings. Establishes the observable set of broadcasters intersecting the target district.

Figure 2 — District Selection

Used district population density vs. 2024 party win likelihood—combined with Redistricting Hub data availability—to narrow and justify which districts could support precinct-level analysis.

Figure 3 — Proof of Concept: Keyword Prevalence

First POC (TX). For each month, compares the percentage of stations whose filings contain selected terms—an interpretable district-wide signal before moving to finer-grained precinct splits.

Figure 4 — Precinct Split and “Lead/Lag” Signal

IL-13. After topic clustering proved noisy, manually consolidated narrative categories tied to community-relevant issues. Precincts were labeled using 2024 election results; the resulting time series suggests a “leading the conversation” pattern between red- and blue-leaning broadcast audiences.

  • Data: FCC Public Information Files (quarterly PDFs) + FCC station metadata + Redistricting Hub precinct shapefiles
  • Methods: PDF extraction → field normalization → geo-joins → time-series aggregation → NLP signals (keywords / manual topic sets)

Graduate School Projects

Monte Carlo RL Racetrack Simulation

Developed a Monte Carlo Agent to optimize performance on a simulated racetrack with slippage, using reinforcement learning. The project includes custom track building and performance visualizations.

AI Genetic Algorithm

Implemented a genetic algorithm to solve the backpacking problem, optimizing item selection to maximize total value without exceeding a weight limit of 250 units. Each item is characterized by its weight and importance value.

Sudoku-Solver-CSP

The solver implements constraint satisfaction problem (CSP) principles, ensuring that the puzzle adheres to the fundamental rules of Sudoku. The solver uses recursive backtracking, which tries numbers 1-9 in empty cells and backtracks when it encounters a conflict.

Pre-Graduate School Projects

FM Radio Landscape

FCC data on ownership and geolocation of FM stations, geospatial data for each congressional district, and US Census data combine to create an ‘FM Landscape.’ Useful for targeted messaging strategies by voting district or population centers. Ongoing project in development.

Stellar Object Classification

Developed a random forest classifier using data on light wavelengths and redshift from over 2 million labeled stellar objects, queried via SQL. Achieved 99% accuracy on unseen data.

Ames Housing Project
